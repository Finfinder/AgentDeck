## AgentDeck

A desktop agentic IDE inspired by VS Code. MVP built on Electron, React, Monaco, and Node/TypeScript services with local memory and MCP support.

## Status

Projekt w fazie MVP / greenfield. Plan implementacji i research są utrzymywane w repozytorium AI_Instruction.

## Skład technologiczny MVP

- Electron (desktop shell)
- React + Monaco (renderer i edytor)
- Node/TypeScript (usługi, Runtime, MCP, Extension Host)
- SQLite + sqlite-vec (lokalny indeks i pamięć)
- Tree-sitter (chunking kodu)
